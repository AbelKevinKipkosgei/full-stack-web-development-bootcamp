Projects

PC Builder Model

PC Builder Model is a model to help users build pc's based on their use case and budget. The model also checks for part compatibility to prevent component mismatch. The streamlit user interface allows the user also to pick parts or automate builds.Google Colab and Pandas library was used for data cleaning, feature engineering, and training models using regression and random forest algorithms for price prediction. The model is deployed on Streamlit.

UberFundi

UberFundi is a platform that helps users to find reliable and skilled service providers based on their location. NextJS framework was used for both frontend development and backend development Clerk for authentication and PostgreSQL and Neon for database development and deployment. The web app is deployed on Vercel.

UrbanMart E-commerce Site

UrbanMart is a one stop online shopping platform for gamers offering a wide range of products from gaming pcs and laptops to gaming accessories such as headsets. React library was used for frontend development, Flask for backend and PostgreSQL for database development. The web app is deployed on Render.
